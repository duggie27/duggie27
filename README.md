### 😶‍🌫️ Project Detail
- 주제 : AI기반 돼지 객체 자세 분석 서비스
- 프로젝트명: 피그메이트 (PigMate)
- 참여기간 : 2024.05.27 ~ 2024.06.20 (4주)
- 수행 역할 : 프론트엔드 총괄, Yolov7 객체 탐지 모델 학습
- 프로젝트 링크 : https://github.com/2023-SMHRD-KDT-AI-16/PIGMATE_repo

### 주요 개발 내용 
- 돼지 자세와 각종 환경 데이터를 기준으로 돼지의 질병을 예측한다.
- 환경기준을 시각화
- 한돈 뉴스 페이지 구현
- 이상치 알림 ( 이메일 API )
- 오늘의 리포트


### 문제 해결 경험 
- Yolov5부터 Yolov7까지 모델을 선정하는 데 있어서 애를 먹었다. 사실 처음에는 무슨 차이가 있을까 싶었는데 알고보니 v5는 경량화된 모델이 있다고 하더라.
- 아무튼 딥러닝은 데이터의 양이 중요한 듯 하다. 충분한 데이터를 가지고 있다면 학습시에 도움이 되는 듯 싶다.
- 우리는 다시 라벨링을 했다. 사진 800장 , 정답데이터 약 4500장.


### 느낀 점
- 처음으로 프론트 단을 혼자 맡아서 해 보았는데 성취감 장난 아니다. 게다가 모델까지 열심히 했다. 팀 내에서 민폐가 되지 않으려고 노력했고 1인분을 넘어서 1.5인분까지 해보려 노력했다. 프로젝트의 완성도 또한 만족스럽고, "돼지"라는 주제 자체가 신선하고 좋았다.
- 아쉬웠던 점: 프로젝트를 진행할수록 코드가 너무 복잡하고 길어졌다. index.jsp에는 style태그랑 script 태그랑 html태그까지 정확히 분리되지 못했다. 그것때문에 유지보수가 어려웠던 경험이 있다.
- 개선방안 : 각종 IoT 환경데이터를 수집하여 실시간으로 반영한다면 정말 좋을 것 같다.  코드에 관해서는 객체지향으로.
